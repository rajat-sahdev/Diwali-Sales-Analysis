# 🎇 Diwali Sales Analysis

## 📖 About the Project
This is my second data analytics project, where I performed Exploratory Data Analysis (EDA) on a Diwali Sales dataset using Python. The objective was to understand customer purchasing behavior and identify key business insights through data visualization.

## 🎯 Objectives
- Clean and preprocess the dataset
- Analyze customer demographics
- Identify top-performing states and occupations
- Find high-selling product categories
- Visualize insights using charts

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
The dataset contains customer details such as:
- Gender
- Age Group
- State
- Occupation
- Product Category
- Purchase Amount

## 📊 Key Insights
- Female customers contributed more to overall sales.
- Customers aged 26–35 made the highest purchases.
- Uttar Pradesh, Maharashtra, and Karnataka generated the highest sales.
- Married women were among the major buyers.
- IT, Healthcare, and Aviation professionals had higher purchasing power.
